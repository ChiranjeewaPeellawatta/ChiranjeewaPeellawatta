Hi there! <br />
I’m **Chiranjeewa Peellawatta**, a passionate **Software Engineering** undergraduate from **Sri Lanka**! 🎓

I aim to become an expert in **Full Stack Development**, particularly focusing on the **MERN stack**. 🚀
